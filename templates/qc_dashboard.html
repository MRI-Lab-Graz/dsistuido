Quick Guide
One box+strip plot per QC metric (DSI Studio SRC/FIB + QSIPrep), every subject/session as a point - the same idea as MRIQC's group report. Severity (severe/moderate/mild) is graded against this cohort's own distribution, so a project with a systemic problem still shows its real spread here even if nothing stands out as an outlier in the per-file flags. Hover a point for its subject/session and exact value. Run "QA / QC" on the Pipeline page first (with a QSIPrep directory set, to include those metrics too).
Set an output directory and click "Load metrics".
