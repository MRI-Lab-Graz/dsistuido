### 2. FIB FILE VALIDATION (validate_fib_file method)
   Checks after EVERY FIB generation:
   
   ✓ File exists on disk
   ✓ File size > 1 MB (detects empty/incomplete files)
   ✓ File structure integrity (scipy.io.loadmat)
   ✓ Presence of expected diffusion metrics
   ✓ Metric count reported in log
   
   Example validation output:
   ✓ FIB validated: sub-1293171_ses-3.odf.gqi.fz (62.45 MB, metrics: 8)
   
   Or on failure:
   ✗ FIB validation failed: sub-1293175_ses-1.odf.gqi.fz
     → File is suspiciously small (0.45 MB), may be corrupt

### 3. SRC FILE VALIDATION
   Checks after SRC generation:
   
   ✓ File exists on disk
